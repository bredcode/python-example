### 문제

정수로 이루어진 리스트와 하나의 정수 `k`가 주어집니다. 이 리스트를 `k`의 값에 따라 부분적으로 역순으로 정렬해야 합니다. 구체적으로, 리스트의 처음부터 `k`개의 요소를 역순으로 정렬하고, 다음 `k`개의 요소를 역순으로 정렬하는 과정을 리스트 끝까지 반복합니다. 만약 리스트의 마지막 부분에서 `k`개보다 적은 요소가 남아있다면, 남은 모든 요소를 역순으로 정렬하세요.

#### 입력 예시:

[1, 2, 3, 4, 5, 6, 7, 8, 9]

k: 3

#### 출력 예시:

[3, 2, 1, 6, 5, 4, 9, 8, 7]
